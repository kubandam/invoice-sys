## Setup
Install dependencies
```
$ yarn install
```
Copy .env.exmaple and rename to .env

Create mongoDB localhost database
> mongodb://127.0.0.1:27017/invoice-sys

## Build
```
$ yarn dev
```
Open
> http://localhost:3000/

Try create a user to test the database
